# Optimized "About Me" Page

**My about me** page was created entirely with TypeScript, which I preferred to use in this project because of these main reasons:
- Better optimization
- Better readability

I also didn't want to use these big frameworks :)

---

# Technology stack used

- **TypeScript** - The main Superset of this project, logic (most used)
- **HTML** - Webpage Language (least used)
- **CSS** - Customization Style Sheet
- **TAILWINDCSS** - utility-first CSS framework

---

# Project layout

```
wiktor-ww/about-me
├── README.MD          - This file
├── index.html         - Main page file
├── /public            - Images
└── /src               - Source code
    ├── App.mts        - The main page return file.
    ├── render.mts     - The page render system
    ├── style.css      - The design file which customises the default page design.
    ├── tippy.mts      - Tippy library used for displaying tooltips on the website
```
